# Project Overview

This repository documents my hands-on Microsoft Intune Administration laboratory built within a Microsoft cloud environment.

The project demonstrates practical endpoint management skills by configuring Microsoft Intune from the ground up, enrolling real devices, deploying applications, configuring security policies, monitoring device compliance, and troubleshooting common administrative scenarios.

Rather than simply describing Microsoft Intune features, this portfolio documents the complete administrative workflow used by IT Support Engineers, Microsoft 365 Administrators and Endpoint Administrators in real-world environments.

---

# Project Objectives

The objectives of this project are to:

- Learn Microsoft Intune administration through hands-on practice
- Understand Mobile Device Management (MDM)
- Manage Android devices using Microsoft Intune
- Configure security and compliance policies
- Deploy and manage applications
- Perform endpoint administration
- Monitor managed devices
- Troubleshoot common Intune issues
- Produce professional documentation suitable for a GitHub portfolio
